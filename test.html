📦 1. Box Model (Kotak CSS)

Setiap elemen HTML di CSS dianggap sebagai kotak yang terdiri dari 4 lapisan:

MARGIN (Luar)
BORDER
PADDING (Dalam)
CONTENT

Ini adalah konten sebenarnya
(teks, gambar, video, dll)

💡 Analogi Sederhana:

Bayangkan sebuah foto (content) yang ditempel di bingkai (padding), dikelilingi garis tepi (border), dan ditempatkan di meja dengan jarak dari dinding (margin).

/* Box Model dalam CSS */ .kotak { margin: 30px; /* Jarak LUAR elemen */ border: 5px solid red; /* Garis tepi */ padding: 40px; /* Jarak DALAM elemen */ content: "Teks..."; /* Isi sebenarnya */ }

⚖️ 2. Perbedaan Margin vs Padding

PADDING (Ruang DALAM)
🎨 Background Biru sampai ke tepi!
Padding = 30px
/* Padding */ .box { background: blue; padding: 30px; /* Ruang 30px di DALAM elemen, ikut warna biru */ }

✅ Gunakan Padding untuk:

  • Memberi "napas" pada konten
  • Memperbesar area klik tombol
  • Jarak teks dari border
MARGIN (Ruang LUAR)
📏 Background Oranye hanya konten!
Margin = 30px
/* Margin */ .box { background: orange; margin: 30px; /* Ruang 30px di LUAR elemen, transparan */ }

✅ Gunakan Margin untuk:

  • Jarak antar elemen
  • Memusatkan elemen (margin: auto)
  • Mendorong elemen menjauh

🔍 Perbedaan Kunci:

Aspek Padding Margin
Lokasi Di dalam border Di luar border
Warna BG Ikut warna elemen Transparan (tidak terlihat)
Nilai negatif? ❌ Tidak bisa ✅ Bisa (menarik elemen)

🌊 3. Float & Clear

Float membuat elemen "mengambang" ke kiri/kanan, memungkinkan konten mengelilinginya (seperti teks mengelilingi gambar di koran).

Contoh Float:

FLOAT: LEFT
🏊‍♂️
FLOAT: RIGHT
🏊‍♀️

Teks ini mengalir di sekitar elemen yang difloat. Float digunakan untuk membuat layout multi-kolom pada zaman sebelum Flexbox ada. Namun, float memiliki masalah: parent elemen akan "runtuh" (collapsed) jika semua children-nya difloat.

CLEAR: BOTH → Elemen ini turun ke bawah, membersihkan float!
/* Float */ .gambar { float: left; /* atau right, none */ margin-right: 20px; } /* Clear - Membersihkan efek float */ .footer { clear: both; /* left, right, both */ } /* Clearfix - Solusi modern untuk parent yang collapsed */ .parent::after { content: ""; display: table; clear: both; }

⚠️ Masalah Float:

Ketika semua elemen child menggunakan float, parent elemen kehilangan tinggi (height menjadi 0). Solusinya: gunakan clearfix atau overflow: hidden pada parent.

🎯 4. Flexbox Layout

Flexbox adalah sistem layout 1 dimensi (baris ATAU kolom) yang lebih modern dan fleksibel daripada float.

A. Flex Direction: Row (Horizontal)

Item 1
flex: 1
Item 2
flex: 1
Item 3
flex: 1

B. Flex Direction: Column (Vertikal)

Item Atas
Item Tengah
Item Bawah


C. Justify Content: Center (Penjajaran Horizontal)

🎯 Center!
/* Flexbox Container */ .container { display: flex; /* Aktifkan flexbox */ flex-direction: row; /* row | column */ justify-content: center; /* horizontal: flex-start | center | flex-end | space-between | space-around */ align-items: center; /* vertikal: stretch | flex-start | center | flex-end */ gap: 20px; /* Jarak antar item */ flex-wrap: wrap; /* wrap | nowrap (turun ke bawah jika penuh) */ } /* Flex Items */ .item { flex: 1; /* Bagi ruang sama rata */ flex: 2; /* 2x lebih lebar dari flex: 1 */ }

✅ Keunggulan Flexbox:

  • Vertical centering sangat mudah (tidak mungkin dengan float)
  • Item otomatis sama tinggi (equal height columns)
  • Urutan item bisa diubah tanpa mengubah HTML
  • Responsif dengan mudah menggunakan flex-wrap

📰 5. Multiple Column Layouts

Ada 3 cara membuat layout multi-kolom: Float (tradisional), Flexbox (modern), dan Grid (paling powerful).

1. FLOAT (Tradisional)
Kolom 1
33.33%
Kolom 2
33.33%
Kolom 3
33.33%
.col { float: left; width: 33.33%; } .clearfix::after { content: ""; clear: both; display: table; }
❌ Kekurangan: Perlu clearfix, sulit equal height, tidak responsif otomatis.
2. FLEXBOX (Modern)
Kolom 1
flex: 1
Kolom 2
flex: 1
Kolom 3
flex: 1
.container { display: flex; gap: 10px; } .col { flex: 1; /* Bagi ruang */ }
✅ Keunggulan: Equal height otomatis, mudah diatur gap, responsif.
3. CSS GRID (Powerful)
Kolom 1
Kolom 2
Kolom 3
.container { display: grid; grid-template-columns: 1fr 1fr 1fr; gap: 10px; }
🚀 Terbaik untuk: Layout 2D kompleks, masonry layout, area template.

📊 Perbandingan Cepat:

Fitur Float Flexbox Grid
Dimensi 1D (susah) 1D (baik) 2D (terbaik)
Equal Height ❌ Sulit ✅ Otomatis ✅ Otomatis
Vertical Center ❌ Tidak bisa ✅ Mudah ✅ Mudah
